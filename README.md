# Cryptocurrency Price Tracker and Paper Trading App
## SwiftUI + Combine + MVVM (Work in progress)

## Introduction

Welcome to the GitHub repository for my innovative cryptocurrency application. This app is designed to provide real-time price tracking of various cryptocurrencies in USD and offers a feature-rich environment for cryptocurrency enthusiasts to manage their virtual portfolio. My goal is to evolve this application into a comprehensive paper trading platform for cryptocurrencies.

## Features

- Real-Time Price Updates: Fetches the latest prices in USD for a wide range of cryptocurrencies.
-  Portfolio Management: Allows users to add and track their desired cryptocurrencies in a personal portfolio.
-  User-Friendly Interface: A clean and intuitive interface ensures a seamless user experience.
- [Upcoming] Paper Trading Functionality: A simulated trading environment for users to practice trading strategies without financial risk.
